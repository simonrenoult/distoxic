Utilisation - Import / Export d'un projet


DisToxic vous permet d'exporter et d'importer indéfiniement des projets afin de les rendre portable (pour une utilisation distante). Ainsi, il vout suffit de vous aider du menu ou de la barre d'outils :

L'import d'un projet


Vous avez le choix parmi 2 modes d'import :

  • l'archive : projet compressé seulement en .zip.
  • le répertoire classique : projet stocké dans un répertoire.
l'importation du projet se fera ensuite automatiquement des lors que vous aurez sélectionné un chemin valide pour votre archive/dossier.


L'export d'un projet


Tout comme l'importation de projet, Distoxic prévoit les 2 mêmes modes. Des que vous aurez sélectionné un chemin valide, l'exort se fera automatiquement.
