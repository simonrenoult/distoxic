Utilisation - Enregistrement d'un projet


DisToxic vous permet d'enregistrer rapidement vos données grâce au deux fonctionalités présentes dans la barre d'outils / menu.


vous pouvez enregistrer les tableaux séparément tout comme par triplet (au maximum les 3 tableaux d'un onglet). Chaque enregistrement déclenchera la création d'un nouveau fichier, rattaché à son tableau graphique.
Comme le fichier original n'est pas modifié, vous pourrez suivre chronologiquement vos sauvegardes par projet.

Sur ce premier exemple, nous pouvons observer que l'utilisateur a d'abord selectionné le tableau du haut (encadré vert) avant de cliquer sur Enregistrer le fichier. On peut alors voir dans l'arbre de sélection la présence du fichier qui vient d'être créé : il se distingue de l'original par son suffixe (NUMERO).


Cependant on peut enregistrer les 3 tableaux en un clic : Le même processus est alors déclenché pour chaque tableau.
