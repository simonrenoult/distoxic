Utilisation - Introduction

Vous pouvez consulter cette aide à tout moment afin de mieux comprendre les fonctionnalités graphiques du logiciel.
Elle est constituée en 2 parties :

  • Import / export d'un projet explique l'export et l'import d'un projet dans votre espace de travail.
  • Enregistrement d'un projet explique l'enregistrement d'un ou plusieurs tableaux.
  • Ajout / suppression de colonnes dans un tableau explique comment éditer des colonnes dans un tableau de type SDF.
  • Ajout / suppression de lignes dans un tableau explique comment éditer des tableaux.
  • Ajouter des fragments à une molécule explique comment ajouter des fragments aux molécules.
