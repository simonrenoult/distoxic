Utilisation - Ajouter / supprimer des colonnes


DisToxic offre la possibilité d'ajouter et de supprimer des colonnes uniquement sur le tableau représentatif du fichier.sdf.

Dans les deux cas, il suffit de sélectionner le tableau concerné, puis de faire un clic droit.
Un menu contextuel apparaît alors comme dans la capture ci dessous :


Une boîte de dialogue vous demande de saisir le nom de la colonne à ajouter/supprimer.
Le processus demandé est alors effectué.
