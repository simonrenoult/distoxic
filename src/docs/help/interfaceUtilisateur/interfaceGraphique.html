Interface utilisateur - Interface principale


DisToxic est un logiciel facile à prendre en main. L'interface a été optimisé afin d'assurer la meilleur utilisation
Ce chapitre est dédié à l'exploration du logiciel afin de connaître l'ensemble des fonctionnalités prpoposées.

On peut distinguer 4 zones graphiques composant l'application :

  1. Le menu.
  2. La barre d'outils.
  3. L'arbre de sélection.
  4. La zone d'édition des données. Chaque tableau de données représente un fichier ouvert.
