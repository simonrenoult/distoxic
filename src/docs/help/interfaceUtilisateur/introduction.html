Interface utilisateur - Introduction

Vous pouvez consulter cette aide à tout moment afin de mieux comprendre les fonctionnalités graphiques du logiciel.
Elle est constituée en 4 parties :
  • Interface principale présente globalement le logiciel .
  • Barre d'outils présente la barre d'outils du logiciel.
  • Arbre de Sélection présente l'arbre de sélections lié à l'espace de travail du logiciel.
  • Menu présente le menu de DisToxic.
