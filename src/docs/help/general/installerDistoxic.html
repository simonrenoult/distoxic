Général - Installation


Le logiciel DisToxic prévoit une installation facile pour tout utilisateur ainsi qu'une riche documentation afin de rendre son utilisation plus confortable.

Installation de DisToxic sur windows

Pour installer Distoxic, il faut double cliquer sur l'exécutable DisToxic_setup.exe et suivre l'assistant d'installation. Après avoir validé la licence du logiciel, vous configurerez l'installation du logiciel.
Dès l'installation terminée, vérifiez que l'ensemble des composant listés ci dessous sont présents :
  • DisToxic.exe : le logiciel.
  • Site Web : page HTML référant au site internet du logiciel.
  • Aide DisToxic : manuel d'aide du logiciel Distoxic
  • Lisez-moi : fichier qui décrit rapidement le logiciel.
  • uninstall : permet la désinstallation du logiciel.
  • distoxic : dossier contenant le projet du logiciel DisToxic.

Utilisation de distoxic sur tous les systèmes d'exploitation

Dans le cardre d'une utilisation multi-plateforme, DisToxic présente une version portable en JAVA testé sur les systèmes d'exploitation Windows, Mac OS X et GNU/Linux équipés d'un environnment JAVA (JRE). Vous pouvez télécharger la dernière version du JRE :

  • pour Windows.
  • pour Mac OS X.
  • pour GNU/Linux
